#  &Pizza - Restaurant Landing Page

A responsive pizza restaurant landing page with custom CSS and JavaScript form validation.

##  Live Demo
[Click here to view](https://alishahwaiz-182.github.io/Pizza-clone/)

##  Features
- Responsive navbar with hamburger menu
- Full screen hero banner
- Signature pizza menu section with cards
- Contact form with SweetAlert2 validation
- Responsive footer with quick links

##  Built With
- HTML5
- CSS3 (Custom - no Bootstrap)
- JavaScript
- SweetAlert2
